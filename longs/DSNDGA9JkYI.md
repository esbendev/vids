[ver el video](https://www.youtube.com/watch?v=DSNDGA9JkYI)

## descripción:

En este video, exploramos el emocionante mundo del open source (código abierto) y te muestro cómo realizar tu primera contribución. Es una guía para principiantes que abarca desde la creación de una cuenta en GitHub hasta el aporte final. Aprende sobre proyectos open source y el coding necesario para participar, todo con un enfoque práctico y visual. ¡No te pierdas este tutorial de git para empezar tu viaje en la programación de código abierto!

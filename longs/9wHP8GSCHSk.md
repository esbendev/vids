# Ventoy es PERFECTO para probar distros - Te muestro cómo

[ver el video](https://www.youtube.com/watch?v=9wHP8GSCHSk)

## descripción:

En este video, les muestro cómo usar Ventoy, una herramienta para crear un "multiboot usb". Con Ventoy, puedes probar diferentes distros de linux sin tener que formatear la tu pendrive cada vez. Aprende cómo usar Ventoy para simplificar tus pruebas de sistemas operativos.
